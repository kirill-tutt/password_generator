# password_generator
Я представляю свой проект по генерации сложных паролей для аккаунтов. Продукт может по умолчанию создавать пароли, которые невозможны для взлома при помощи метода перебора. Все пароли и логины сохраняются в отдельный файл .txt (в дальнейшем обновлении я добавлю шифрование файла так, чтобы только пользователь, чьи пароли хранятся файле, смог его читать, а случае слива в сеть всего файла, там были бы разного рода непонятное сочетание символов). В файле хранится время и дата записи пароля; название сайта; логин и пароль в удобном оформлении. Весь текст написан на русском и на английском языке. Исходный код также понятно расписан и объяснён при помощи комментариев. Буду рад, если вы воспользуетесь моей программой)
---------------------------------------------------------
I present my project for generating complex passwords for accounts. The product can, by default, create passwords that are impossible to crack using brute-force methods. All passwords and logins are saved in a separate .txt file (in a future update, I'll add file encryption so that only the user whose passwords are stored in the file can read it, and if the entire file were leaked, it would contain all sorts of incomprehensible character combinations). The file stores the time and date the password was saved, the website name, and the login and password in a convenient format. All text is written in Russian and English. The source code is also clearly written and explained with comments. I'd be happy if you used my program)
